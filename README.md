# 💬 Mini Chatbox (React + Flask)

## 📌 About the Project
This is a simple full-stack chatbox application where users can send messages and receive AI responses.  
The frontend is built with React and the backend is built with Flask.

---

## ⚙️ Tech Stack
- Frontend: React
- Backend: Flask (Python)
- API: LLM API (e.g. Groq/OpenAI)
- Communication: REST API

---

## 🚀 Features
- Send and receive messages
- Real-time chat interface
- Backend API integration
- Simple and clean UI
